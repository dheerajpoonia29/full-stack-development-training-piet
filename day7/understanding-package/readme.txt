1. Module is a JavaScript file which contains functions 
    - we export a function by syntax 
    - module.exports = {functionName}
    - to import module we use require method and pass our module file path
2. Package is a group of module
    - for example react is a package 
    - useState, useEffect are module of react package
3. Library is group of package
4. so React is a Library which used to build UI or frontend
